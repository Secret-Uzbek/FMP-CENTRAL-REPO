# FMP CENTRAL REPOSITORY

## Цель
Централизовать все репозитории FMP в одной фрактальной структуре.

## Структура
- NULLO: ядро и живые протоколы
- PLT: экосистемы и приложения
- UCOMM: аудитории и коммуникации
- FMP: теория и публикации
